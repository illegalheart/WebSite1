このWebサイトができるまで

Weser-ITのWebサイトは、最新のAI技術と自動化ツールを駆使して構築されました。
「手書きとFAXを減らす」という理念を自ら体現する、その開発プロセスをご紹介します。

1. AIとの協働開発

このサイトのコード（HTML/CSS/JS）は、AIアシスタントと共に書かれました。
デザインのコンセプト立案から、画像の生成まで、AIがクリエイティブパートナーとして参加しています。

  • Vite + Vanilla JSによる高速な構成
  • AIによるオリジナル画像生成
  • 対話形式での迅速な修正と改善

2. インフラのコード化 (IaC)

サーバー環境を手動でポチポチ設定することはありません。
Terraform というツールを使い、Azure上のインフラ構成を「コード」として記述・管理しています。

resource "azurerm_static_site" "web" {
  name = "stapp-weser-it-web"
  location = "eastasia"
  sku_tier = "Free"
}

3. 完全自動デプロイ (CI/CD)

コードを修正して保存（Git Push）するだけで、あとは全自動。
GitHub Actions が変更を検知し、ビルドからAzureへの公開までを数分で完了させます。

  • 人為的なアップロードミスの排除
  • 24時間365日、いつでも即時公開
  • 開発に集中できる環境作り

この「仕組み」そのものをご提案します

私たちWeser-ITは、こうした自動化技術を、お客様の現場業務にも応用します。
複雑なことは裏側で処理し、表側はシンプルに。
